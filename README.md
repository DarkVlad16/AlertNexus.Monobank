# AlertNexus.Monobank

Статична сторінка для формування посилання на донат через Monobank для AlertNexus.

## Використання

Додайте ID банки Monobank до адреси сторінки:

```text
https://darkvlad16.github.io/AlertNexus.Monobank/?jar=ID_БАНКИ
```

Донатер вказує нік, суму та повідомлення. Сторінка відкриває Monobank із параметрами:

- `a` — сума донату;
- `t` — коментар у форматі `@нік повідомлення`.

AlertNexus отримує транзакцію через Monobank API, використовує нік і повідомлення з коментаря та не показує банківське ім’я відправника.

Monobank дозволяє до 160 символів у коментарі. Сторінка враховує нік, пробіл і повідомлення в одному спільному ліміті та показує лічильник символів.

Сторінка не має backend, не використовує базу даних і не зберігає введені дані.
Вибрана мова інтерфейсу зберігається локально в браузері.
